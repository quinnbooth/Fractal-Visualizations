Julia Set
Similar to the Mandelbrot Set, the Julia Set is determined by the iterative behavior of the function f(z) = z^2 + c. If, for a given constant 'c', the initial complex number 'z' does not diverge to infinity when iterated, then 'z' is considered to be part of the Julia Set.
