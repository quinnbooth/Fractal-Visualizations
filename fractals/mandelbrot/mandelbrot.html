Mandelbrot Set
The Mandelbrot set is defined by the behavior of complex numbers computed iteratively using the function f(z) = z^2 + c. If a complex number 'c' does not lead the iterated values of 'z' to diverge to infinity from the starting point z = 0, it belongs to the Mandelbrot set.
